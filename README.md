# MedGrab

Place .env in backend folder

Setting up kong:
cd backend
1. Run: chmod +x kong-basic-setup.sh
2. Run: ./kong-basic-setup.sh
Or just run the bat file

To run backend:
cd backend
docker compose up -d --build 

To run frontend:
cd frontend
npm i 
npm run dev
